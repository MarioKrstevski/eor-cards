Original text:

Convert the following into numbered Anki Cloze cards formatted for Excel. Output exactly four columns separated by a single pipe | character with no spaces before or after each pipe. Each row must contain exactly three pipe characters total. Column 1 must contain the card number only. Column 2 must contain the cloze card text only. Column 3 must contain the additional context field text only. Column 4 must contain the tag field only. Do not include any additional pipe characters anywhere in the output. Do not include blank lines or any extra text before or after the cards. If no additional context exists for that card, leave Column 3 blank. The tag field must follow the format: PA Doctrine::EOR::Emergency Medicine::Orthopedics/Rheumatology::[Blueprint Topic]. The final segment must be the exact blueprint topic name (normalized only for spacing, e.g., spaces → underscores). Do not use subtopics or inferred categories in the tag.
